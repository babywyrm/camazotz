Data Flow

How the Camazotz stack connects to the identity provider. Only the oauth_delegation and revocation labs make live HTTP calls. The RBAC lab merges group claims from environment variables.

Provider Selection and Degradation

Two levels of degradation. Gateway-level: IdP unreachable, entire runtime uses mock. Per-tool: IdP reachable but a specific HTTP call fails, lab falls back to synthetic token.

Token Lifecycle

The three live IdP HTTP operations and which lab triggers each. All use form-encoded POST with client_id/client_secret auth and 5s timeout.
