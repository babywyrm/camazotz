Provider Selection and Degradation

Two levels of degradation. Gateway-level: ZITADEL unreachable, entire runtime uses mock. Per-tool: ZITADEL reachable but a specific HTTP call fails, lab falls back to synthetic token.

Token Lifecycle

The three live ZITADEL HTTP operations and which lab triggers each. All use form-encoded POST with client_id/client_secret auth and 5s timeout.
